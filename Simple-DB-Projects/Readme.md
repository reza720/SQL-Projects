These are simple database projects created for practicing database design and SQL. The projects are inspired by exercises from AssignmentDude and are intended to improve skills in:
- Database schema design  
- Writing SQL queries  
- Understanding relationships between tables  

Each project includes:
- Project description in the README file (directly taken from the AssignmentDude website)  
- `schema.sql`  
- `seed-data.sql`  
- `sample-queries.sql`
